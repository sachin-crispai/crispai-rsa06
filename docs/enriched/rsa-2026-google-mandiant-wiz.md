#### New Product Announcements & Updates

**Google Security AI Announcements**
Google Cloud is debuting a suite of AI-powered security innovations aimed at transitioning organizations toward an "Agentic SOC" (Security Operations Center).

*   **AI Agent "Triage and Investigation"**: Embedded in Google's "Security Operations" platform, this AI agent autonomously reviews security alerts, gathers additional data and context, and assesses threats to reduce false positives for security analysts. It is currently available in public preview.
*   **AI-Powered Dark Web Analysis**: Built into Google Threat Intelligence, this new tool utilizes Gemini's models to crawl the dark web, processing 8-10 million posts daily with 98% accuracy to identify relevant threats for organizations.
*   **Custom Security Agents**: Google Security Operations customers can now build their own enterprise-ready security agents with remote Model Context Protocol (MCP) server support, which will be generally available in early April.
*   **Network Security Integration**: The in-band mode is now generally available, allowing customers to secure application workloads using third-party network appliances without altering existing routing policies or network architecture.
*   **Other Innovations**: New security innovations are also being rolled out across Chrome Enterprise, Security Command Center, and network management.

**Mandiant Threat Intelligence Updates**
Mandiant, a Google cybersecurity subsidiary, has released its M-Trends 2026 report and a special report on AI risk and resilience.

*   **M-Trends 2026 Report**: Key findings indicate that attackers are operating faster and more collaboratively, increasingly targeting systems organizations rely on for recovery. The median dwell time for attackers increased to 14 days in 2025, up from 11 days the previous year. Voice phishing has surged to become the second most common initial infection vector (11%), surpassing email phishing (6%).
*   **AI Risk and Resilience Report**: This report details how adversaries have transitioned from experimental AI use to deploying adaptive tools and autonomous agents capable of rewriting their own code in real-time.
*   **Trivy Supply-Chain Attack**: Mandiant Consulting CTO Charles Carmakal highlighted that over 1,000 cloud environments have been infected by the Trivy supply-chain attack, with the attackers collaborating with notorious extortion groups like Lapsus$.
