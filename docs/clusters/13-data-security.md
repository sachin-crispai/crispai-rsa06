# Cluster 13 — Data Security & Privacy

**Stack:** Know where your data is, who can touch it, and protect it everywhere it lives.

Data security converges data discovery, classification, access governance, DLP, and privacy compliance into a unified posture. Critical for GDPR, CCPA, HIPAA, and AI data governance.

## Technology Landscape

### Data Discovery & Classification
Find sensitive data across structured and unstructured sources — cloud, on-prem, SaaS.
- **Leaders:** BigID, Varonis, Securiti, Microsoft Purview

### Data Security Posture Management (DSPM)
Continuously assess where sensitive data is exposed, who has access, and remediate.
- **Leaders:** Varonis, BigID, Securiti, Sentra, Normalyze

### DLP (Data Loss Prevention)
Prevent unauthorized exfiltration of sensitive data via email, web, USB, cloud.
- **Leaders:** Forcepoint, Zscaler, Netskope, Microsoft Purview DLP

### Encryption & Key Management
Protect data at rest and in transit; manage encryption keys at enterprise scale.
- **Leaders:** Thales CipherTrust, Entrust, IBM Guardium Encryption

### Privacy Compliance (GDPR/CCPA)
Automate data subject rights, consent management, privacy impact assessments.
- **Leaders:** BigID, Securiti, OneTrust, TrustArc

## Key Questions to Ask at Booths
- How do you handle unstructured data — PDFs, emails, Slack messages, SharePoint?
- AI governance: can you track what training data has what PII/sensitive content?
- DSPM + CSPM integration — same platform or separate?
- How do you enforce data residency for multi-region cloud deployments?
- Data lineage: can you trace where a sensitive record has traveled?
