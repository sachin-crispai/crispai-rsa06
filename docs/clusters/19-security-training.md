## Technology Landscape

### Phishing Simulation
Send simulated phishing emails; measure click rates; deliver just-in-time training.
- **Leaders:** KnowBe4, Proofpoint, Cofense, Hoxhunt

### Security Awareness Platforms
Course libraries, compliance tracking, policy acknowledgment, LMS integration.
- **Leaders:** KnowBe4, Proofpoint, Ninjio, Curricula

### Human Risk Management
Quantify per-employee risk scores based on behavior, training completion, simulation results.
- **Leaders:** Living Security, KnowBe4 PhishER+, Proofpoint

### Technical Security Training
Hands-on labs for security engineers, developers, and SOC analysts.
- **Leaders:** SANS, Immersive Labs, Hack The Box, TryHackMe, Pluralsight

### Developer Security Training
Teach secure coding practices — embed security in developer workflow.
- **Leaders:** Secure Code Warrior, Snyk Learn, Checkmarx Codebashing

## Key Questions to Ask at Booths
- How do you personalize training based on individual risk behavior?
- Phishing simulation: how do you avoid "crying wolf" with excessive campaigns?
- Integration with HR/LMS systems for compliance reporting?
- How do you measure actual behavior change vs. completion rates?
- AI-generated phishing content in simulations — are you using it?
