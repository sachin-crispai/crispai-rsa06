## Technology Landscape

### EDR (Endpoint Detection & Response)
Continuous endpoint monitoring, threat hunting, automated response.
- **Leaders:** CrowdStrike Falcon, SentinelOne, Microsoft Defender for Endpoint, Carbon Black

### XDR (Extended Detection & Response)
Correlate signals across endpoint, identity, cloud, email, network for unified detection.
- **Leaders:** CrowdStrike XDR, SentinelOne Singularity, Microsoft Defender XDR, Palo Alto Cortex XDR

### EPP (Endpoint Protection Platform)
Traditional AV/NGAV — file-based and behavioral malware prevention.
- **Leaders:** CrowdStrike, SentinelOne, Fortinet, ESET, Microsoft

### Application Control / Allowlisting
Block unauthorized executables — strong posture for ransomware defense.
- **Leaders:** ThreatLocker, Carbon Black App Control, Airlock Digital

### Unified Endpoint Management + Security
Combine device management (MDM/UEM) with security telemetry.
- **Leaders:** Tanium, Microsoft Intune + Defender, Jamf (Apple-focused)

## Key Questions to Ask at Booths
- How do you handle detection latency — time from execution to alert?
- What's your AI/ML model for behavioral detection — cloud vs. on-device inference?
- How does your XDR ingest third-party data sources (not just your own agents)?
- Autonomous response capabilities — what can you remediate without human approval?
- How do you handle macOS and Linux coverage vs. Windows?
