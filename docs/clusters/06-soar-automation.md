## Technology Landscape

### Traditional SOAR
Playbook-driven automation triggered by SIEM alerts — case management, ticketing, enrichment.
- **Leaders:** Splunk SOAR, Palo Alto XSOAR, IBM SOAR

### No-Code / Low-Code Automation
Visual workflow builders — enable non-engineers to build security automations.
- **Leaders:** Tines, Torq, Swimlane

### Agentic Security Operations
LLM-powered agents that autonomously investigate alerts, gather context, and recommend/take actions.
- **Emerging:** CrowdStrike Charlotte AI, SentinelOne Purple AI, Darktrace ActiveAI

### Hyperautomation
End-to-end automation connecting security tools, IT ops, HR, and business workflows.
- **Leaders:** Torq, Tines, ServiceNow Security Operations

## Key Questions to Ask at Booths
- How do you handle hallucination/false confidence in AI-driven automation?
- What audit trail exists for autonomous actions taken without human approval?
- How do you measure MTTR (Mean Time to Respond) improvement?
- What integrations exist out-of-the-box vs. custom connectors required?
- Pricing model — per playbook run, per alert, per seat?
