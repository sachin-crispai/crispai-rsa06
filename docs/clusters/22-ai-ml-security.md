## Technology Landscape

### AI-Powered Security Tools
Use ML/AI to detect threats, reduce alert fatigue, accelerate SOC operations.
- **Leaders:** Darktrace, CrowdStrike, SentinelOne, Vectra, Exabeam

### LLM Application Security
Protect applications built on LLMs from prompt injection, jailbreaking, data leakage.
- **Leaders:** Lakera, CalypsoAI, Robust Intelligence, Prompt Security

### ML Model Security (MLSec)
Protect ML models from adversarial attacks, model inversion, membership inference.
- **Leaders:** HiddenLayer, Protect AI, Robust Intelligence

### AI Governance & Trust
Audit AI systems for bias, explainability, compliance, and responsible AI use.
- **Leaders:** Securiti, BigID, OneTrust (AI Governance), ModelOp

### Generative AI for Security Operations
LLM-powered SOC assistants — natural language queries, automated investigation, report writing.
- **Leaders:** CrowdStrike Charlotte AI, SentinelOne Purple AI, Microsoft Copilot for Security, Splunk AI

---

## Key Questions to Ask at Booths
- Prompt injection attacks — how do you detect and block them in production LLM apps?
- AI model theft/inversion: what signals indicate adversarial probing of your model?
- Explainability: can your AI detections be explained to an analyst in plain language?
- False positive rates for AI detections vs. traditional rule-based detection?
- AI governance: how do you audit LLM decisions for compliance/regulatory purposes?
