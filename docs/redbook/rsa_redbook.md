## Booth Numbering
- **North Hall booths:** prefix `N-` (e.g., N-5845)
- **South Hall booths:** prefix `S-` (e.g., S-3316)
- **Early Stage Expo kiosks:** prefix `ESE-` (e.g., ESE-19)
- **Standard numeric:** no prefix (e.g., Booth 327)
